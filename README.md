## LTspice Cheatsheet
### commands
> - wire : F3
> - move : F7

### model & subckt
> Spice model :
> - Intrinsic SPICE devices, like Diodes & Transistors
> 
> Spice subckt : 
> - Collection of SPICE devices, like OPAMPS


### .lib & .inc
> .inc : 
> - Include everything in the file, even commands (.ac, .op)
> 
> .lib : 
> - Include models & subckts, safer to use
